# 各省志愿填报规则

> ⚠️ 红线提醒：志愿数量、批次设置、投档规则**每年可能微调**，使用时必须以**考生当年本省官方《招生工作规定/考生指南》**为准。本表是结构性框架，不替代当年官方文件。
> 核实动作：`web_search "<省名> <年份> 普通高校招生 志愿设置 考生指南"`，认准省考试院官方发布。

## 两大投档模式（先判断考生省属于哪类）

### 模式 A：院校专业组 + 平行志愿
- **志愿单位 = 院校专业组**（同组内专业选科要求相同，调剂仅在组内）。
- 一所高校可设多个专业组；同组内有"是否服从调剂"选项。
- 采用省份（约 21 个）：北京、天津、上海、江苏、湖北、湖南、广东、福建、河北、辽宁(部分)、重庆、湖南等新高考"3+1+2 / 3+3"地区（具体以当年为准）。
- **推演要点**：必须区分到**专业组**位次，不能只看院校最低位次（见 methodology 专业组分差 + 中外合作组陷阱）。

### 模式 B：专业（类）+ 院校 + 平行志愿
- **志愿单位 = 一个专业(类)+一所院校的组合**（无校内调剂，所见即所得）。
- 采用省份（约 7 个）：浙江、辽宁(部分)、山东、河北(部分)、重庆(部分)、广西(部分新高考)等（以当年为准）。
- 浙江：分段填报，志愿数极多（80~112 个量级）。山东：96 个专业+院校志愿（常见量级）。
- **推演要点**：无校内调剂 → 滑档=直接落到下一志愿，冲稳保梯度要更密集；不存在"组内调剂保底"，每个志愿都要够得着对应专业。

### 模式 C：传统文理 + 院校志愿（旧高考存量）
- 仍未进新高考的少数地区按文/理科 + 平行/顺序志愿。
- 顺序志愿省份第一志愿权重极高，冲的代价大，冲档要更保守。

## 志愿数量参考（结构量级，非当年精确值）

| 省/模式 | 本科批志愿数量级 | 单位 |
|---------|----------------|------|
| 北京（院校专业组）| ~30 个院校专业组 | 每组若干专业+调剂 |
| 湖北/湖南/广东（院校专业组）| ~45 个院校专业组 | 每组 6 专业+调剂 |
| 陕西/山西等新高考首年 | ~45 个院校专业组 | 每组 6 专业 |
| 浙江（专业+院校）| ~80 段 / 分段 | 专业+院校 |
| 山东（专业+院校）| ~96 个 | 专业+院校 |
| 贵州（专业类平行）| ~96 个专业(类) | — |

> 数量决定冲稳保的**分配比例**：志愿越多，冲/稳/保可铺得越开（常见 冲20% / 稳50% / 保30% 量级）；志愿越少越要保守。

## 通用规则要点（各省共性）

1. **平行志愿"分数优先、遵循志愿、一轮投档"**：按位次从高到低，依考生志愿顺序检索，投出即不再检索后续。落档只能等征集志愿/下一批次。
2. **服从调剂**：模式 A 不勾服从调剂，分够校线但不够专业 → **退档**（极危险）。**铁律：每个专业组都勾服从调剂。** 模式 B 无校内调剂，不涉及。
3. **批次**：提前批/本科批/专科批 + 各类专项（国家专项/地方专项/高校专项，有资格门槛）。
4. **征集志愿**：批次录取后未满额会征集，落档考生可二次填报（关注省考试院公告时间）。
5. **选科匹配**：新高考专业组有选科要求（如"物理+化学"），考生选科不符无法报该组。

## 用法
1. 先判断考生省属 A/B/C 哪种模式。
2. `web_search` 拉取考生当年本省官方志愿设置文件，核实数量/批次/调剂规则。
3. 按模式调整冲稳保的密度与服从调剂策略（见 methodology-rank-anchor.md）。
