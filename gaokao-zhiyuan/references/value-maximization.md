# 志愿价值最大化策略（优化层，非强制）

> 定位：在**数据真实（红线不可破）**的前提下，帮考生把同一分数/位次的价值**用到极致**。
> 优先级：本策略是"优化建议"，**永远让位于红线**——不得为追求"更好"而采用未验证数据或拔高到"超出能力范围"。先合规，再最优。

## 四大优化方向

### 1. 院校层次最大化（同位次够得着的，选更高排名/更硬招牌）
同一位次区间内常有多所院校可选，优先级排序参考：
- **办学层次**：双一流 > 省重点/有博士点 > 普通公办本科 > 应用型本科。
- **学科招牌**：有**国家级一流本科专业建设点** / **学科评估 B 类以上** / **ESI 前 1% 学科**的院校优先。
- **保研资格 / 硕博点**：有推免资格、博士点的院校，深造通道更顺，长期价值更高。
- **行业认可度**：行业特色校（如原部委直属、央企共建）在对口行业就业有隐性溢价（如车辆工程之于汽车行业）。
- 做法：在稳/保档里，**同位次优先推层次更高、招牌更硬的那一所**；在推荐理由里点明"为什么这所比同位次其他校更值"。

### 2. 就业出口 / 职业确定性最大化（很多家庭最看重的"毕业即稳定就业"）
同位次/同档下，优先**就业前景好、就业渠道稳**的院校+专业组合。四个抓手：

**① 就业前景好的专业方向**
- 看专业的**就业率、对口率、起薪、行业景气度**（来源：院校官网就业质量报告、麦可思数据、教育部专业目录）。
- 优先**国家战略+产业紧缺**方向（如集成电路、人工智能、临床医学、电气、能源、智能制造等），规避连续亮红牌的过剩专业。
- 注意"好就业"要看**该校该专业**的真实出口，不是专业名字好听。

**② 该校该专业的专业排名高**
- 不只看学校综合排名，更看**这个专业在全国/全行业的排名与口碑**——很多双非院校有"单项冠军"王牌专业（如某理工院校的车辆/电气、某医学院的影像/护理），就业认可度甚至超过名校弱势专业。
- 来源：学科评估结果、软科专业排名、国家级一流专业建设点、行业内口碑。
- 做法：出表时点明"这是该校的王牌专业，行业认可度高"。

**③ 极稳定的就业渠道（毕业即签 / 定向培养，尤其央国企）**
- **企业定向班 / 订单班 / 现代产业学院**：毕业直接进合作企业，尤其**央企/国企**共建的（如电网订单班、铁路/轨道交通定向、烟草、能源、军工配套、东风等汽车央企共建专业）。
- **行业特色校的隐性渠道**：原部委/央企直属或共建院校，在对口央国企招聘有"校友网络+定向通道"优势（如电力、铁路、邮电、地质、矿业、农林类院校）。
- 核实门槛：定向/订单班常有**就业协议、定向区域、违约条款**，必须向考生说明清楚再推荐。
- 来源：院校官网招生章程"订单/定向培养"章节、就业质量报告中的"主要就业单位"。

**④ 政策性 / 体制内出口的招生计划（确定性极高的"上岸"通道）**
面向国家战略与基层的定向计划，**毕业即有编制/岗位保障**，对求稳家庭价值极高。务必先核实资格与服务期约定：

| 计划 | 出口/利好 | 门槛/约定 | 主管/来源 |
|------|----------|----------|----------|
| **公费师范生**（部属6校+地方）| 毕业进中小学有编制 | 协议服务期（常6年）、定向省份 | 教育部/省教育厅 |
| **优师专项（中央/地方优师计划）**| 定向中西部脱贫县中小学教师 | 农村/脱贫地区定向、服务期 | 教育部+乡村振兴方向 |
| **农村订单定向免费医学生**| 毕业到乡镇卫生院当全科医生 | 本省农村定向、服务期(常6年) | 卫健委+教育部，"乡村医生"通道 |
| **公费/定向农科生、特岗相关**| 涉农基层岗位 | 农业农村部/乡村振兴相关定向 | 农业农村部/省农业厅 |
| **公安/警校（入警率高的专业）**| 毕业参加联考入警 | 政审体测、入警考试 | 公安院校 |
| **军校 / 定向培养军士**| 军官/军士岗位 | 政审体检、服役约定 | 军队招生 |
| **国家专项/地方专项/高校专项**| 低位次进好校（基层生源倾斜）| 农村户籍学籍等 | 见 cross-verification 陷阱7 |
| **基层就业导向专业**（如乡村振兴相关、基层公务员对口专业）| 对接基层公务员/选调、乡村振兴岗位 | 关注定向选调资格 | 教育部+乡村振兴 |

- 做法：符合资格的，作为**"稳就业/上岸通道"专列**推荐，把"出口+服务期+定向区域"说清楚，由考生权衡自由度 vs 确定性。


### 3. 办学性质优先级（公办 > 民办 > 中外合作）
明确排序（学费与文凭含金量综合）：
```
公办本科  >  民办本科/独立学院  >  中外合作办学
```
- **公办优先**：学费低（通常 4000~6000/年）、社会认可度高。
- **民办/独立学院**：学费高（1.5~3 万/年），同位次下尽量用公办替代。
- **中外合作办学**：学费最高（2~3 万甚至更高），且常以**虚低位次**出现在投档表（见 cross-verification 陷阱 1）——除非考生/家庭明确接受高学费或冲名校跳板，否则**降序排在最后或剔除**。
- 做法：每所院校/专业组**标注办学性质**；当公办与民办/合作同位次竞争时，**默认推公办**，并说明性质差异让用户知情选择。

### 4. 利用信息差挖特殊招生计划（在够得着+有资格的前提下）
特殊计划常因**位次门槛更低 / 关注度低**，是"花更少位次上更好学校/专业"的机会。但**全部有资格门槛，必须先核实考生符合资格**，否则低位次不能算数（呼应 cross-verification 陷阱 7）。

| 特殊计划 | 利好（信息差点） | 资格门槛（必须先核） |
|---------|----------------|--------------------|
| **国家专项计划** | 面向贫困地区，重点高校投放，位次门槛常低于普通批 | 户籍+学籍在实施区域、农村户籍等 |
| **地方专项计划** | 省属重点高校面向本省农村，分数更友好 | 本省农村户籍+连续学籍 |
| **高校专项（农村单招）** | 名校（含 985/211）面向农村，降分录取 | 农村户籍+成绩条件+需提前网上报名 |
| **中外合作组（反向利用）** | 同校用更低位次进，适合只为名校光环/出国跳板 | 接受高学费；务必告知性价比 |
| **预科班 / 民族班** | 少数民族考生降分 | 民族成分、定向区域 |
| **定向招生**（如公费师范/医学、军士） | 降分+就业保障+学费减免 | 接受定向就业协议、体检政审 |
| **校企合作 / 产业学院组** | 部分院校单列、位次略低 | 注意是否额外收费、培养质量 |
| **新增专业 / 首年招生组** | 无往年数据、考生观望→易捡漏 | 无历史位次→风险与机会并存，标注"首年无数据" |
| **征集志愿** | 一批落档后二次机会，常有好校缺额 | 关注省考试院征集公告时间 |
| **中外校区/异地校区** | 同校不同校区位次差 | 注意毕业证是否标注校区 |

做法：
- 主动检索考生所在省**当年的专项计划目录**（省考试院 + 阳光高考平台），核对考生资格。
- 符合资格的，作为**"信息差捡漏"专列**推荐，明确标注利好与门槛。
- 不符合资格的低位次专项组，**不计入考生"够得着"的依据**。

## 与红线的关系（务必恪守）
1. 价值最大化**不得**突破"超出能力范围"红线——再好的学校，位次够不到就是够不到。
2. 挖特殊计划**不得**跳过资格核实——没资格的低位次是镜花水月。
3. 所有"更优"推荐**仍需真实数据支撑**——优化的是选择，不是数据标准。
4. 把性价比/性质/门槛**透明告知用户**，由用户在知情下决策（降低认知负荷 ≠ 替用户隐藏风险）。

## 出表体现
- 推荐理由里点明"价值点"：为何这所比同位次其他校值（层次/招牌/专业排名/就业出口/性质/特殊计划）。
- 点明专业**就业前景与出口**（是否王牌专业、主要就业单位、是否央国企定向渠道）。
- 单列一栏"💼 稳就业/上岸通道"（若有符合资格的定向/订单/体制内计划，标明出口+服务期+定向区域）。
- 单列一栏"💡 信息差机会"（若有符合资格的专项/捡漏点）。
- 办学性质（公办/民办/合作）逐校标注。
