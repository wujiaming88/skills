# 出表前自检清单（踩坑防线）

> 出表前逐条过。任一不过 → 打回。本清单含真实实战血泪教训。

## A. 数据真实性（红线 1/2）
- [ ] 每所院校都有**历年（≥2 年，力争 3 年）**真实录取位次，不是单年拍板
- [ ] 每个位次都带 **来源 + 年份 + 专业组性质**
- [ ] 关键位次经 **≥2 独立源交叉验证**（不一致已标注争议/以官方为准）
- [ ] 所有"未查到/单源待核/争议"已**如实标注**，全表**零杜撰**
- [ ] 没有用估算/推测数字冒充真实录取数据
- [ ] **三要素核验**：每个录取数据的 ⏱时间(目标年份、非2025/2026混标) / 📍省份(考生本省) / 🎓考生类别(物理/历史、批次、组性质) 都对得上

## B. 锚点正确性（红线 3）
- [ ] 考生位次由**当年一分一段表**定位（不是凭分数估）
- [ ] 当年**批次线/本科线已独立核实**（不直接采信任务书/用户口述）
- [ ] 全程用**位次**跨年比较，没有拿分数直接比

## C. 档位推演正确性（红线 5）
- [ ] 冲/稳/保均由**位次差推演**得出（methodology 模型），非凭校名/感觉
- [ ] **无"超出能力范围"伪冲档**（考生位次远低于该校录取线的已剔除）
- [ ] 冲 2~3 / 稳 3~4 / 保 2~3，梯度连续无断层
- [ ] 顺序志愿/专业+院校模式已按 province-rules 调整密度

## D. 陷阱排查（cross-verification 八陷阱）
- [ ] **中外合作组陷阱**：每校用的是**普通组**位次，未被中外合作虚低位次误导；高学费组已单独标注
- [ ] **科类正确**：用的是考生对应科类（物理/历史/文/理）的数据
- [ ] **专业组分差**：区分了"进得了学校"vs"进得了好专业"，没拿校线最低位次承诺好专业
- [ ] **专项计划**：低位次的专项组，考生若无资格未计入"够得着"依据
- [ ] **大小年**：异常年份用三年中位数，未被单年极值误导

## E. 偏好与提醒
- [ ] 用户的**排除项**已全部剔除（如"不要体育/师范"——逐校核对类型）
- [ ] 意向城市/专业偏好已体现在排序/建议里
- [ ] 每档都提醒**勾服从调剂**（模式 A）
- [ ] 已提醒**当年招生计划册发布后复核专业组代号**
- [ ] 附了**数据可信度声明**

## 🩸 实战血泪教训（真实案例，务必内化）

### 教训 1：把"超出能力范围"误当冲档 ⚠️
- 现场：曾把某校（2025 普通组最低 79629 位）当"冲档"放进表里，但考生位次 91723——**比该校录取线还低约 1.2 万名**。
- 错在哪：那不是"冲一冲能上"，是"普通组根本够不到"。唯一够得到的是该校中外合作组（高学费，已被用户排除）。
- 纠正：**冲档 = 考生位次略高于该校录取位次（差距可控）；不是无限拔高。** 发现差距过大立即剔除，宁可冲档只放真够得着的 2 所。

### 教训 2：中外合作组虚低位次误导
- 现场：湖北经济学院 2025 整体最低 473分/110381位（中外合作），普通组实为 504分/83125位，差近 2.7 万名。
- 纠正：一律用普通组位次匹配；看到异常偏低位次先怀疑中外合作/民办/护理单列组。

### 教训 3：本科线数据不核实
- 现场：任务称"2025 湖北物理本科线 435 分"，官方实为 426 分（特殊线 516）。
- 纠正：当年批次线必须独立核实，不采信口述。

### 教训 4：科类张冠李戴
- 现场：某校"504/83125"是历史类校线，物理类实际最低投档 482，差异巨大。
- 纠正：锁定考生科类再取数。

> 核心心法：**宁可少推荐一所真数据院校，绝不多填一个假数据。数据是志愿的生命线。**
