# 多源交叉验证协议 + 经典数据陷阱

> 红线 1/2/4 的落地执行手册。数据是志愿填报的生命线，一个错位次能毁掉一个志愿。

## 交叉验证协议（每个关键数据点强制执行）

### 三步验证法
1. **官方源打底**：先查省考试院/招生信息网官方投档线统计 + 院校官网招生网。这是最高权威。
2. **聚合源印证**：用中国教育在线、掌上高考、大学生必备网等权威聚合至少 1 个，与官方比对。
3. **冲突仲裁**：≥2 源一致 → 采信；矛盾 → 以官方为准并标注争议；只有 1 源 → 标"单源待核"；都查不到 → 标"未查到"，**绝不编造**。

### 每个数据点必带元信息
```
508分 / 82752位 | 2024 | 第C12809组(普通类) | 来源: 学校信息公开网 + 大学生必备网 ✅✅
```
缺来源 / 缺年份 / 缺专业组性质的位次数据 = 不合格，打回重查。

### 可信度标记体系
- ✅✅ 多源官方/权威一致
- ✅ 单一权威源
- ⚠️ 单源待核 / 存在争议
- ❌ / "未查到" 查不到，如实标注

## 🪤 经典陷阱清单（逐个排查）

### 陷阱 1：中外合作组位次陷阱（最高频，必查）⚠️
许多院校"整体最低位次"实际是**中外合作办学组**（学费 2~3 万/年），普通组位次远高于此。
- 反例（实战）：湖北经济学院 2025 整体最低 = 第15组(中外合作) 473分/110381位；**普通组最低 = 第13组 504分/83125位**。差了近 2.7 万名！
- **铁律：一律用普通组位次匹配。** 看到某校位次异常偏低（比同档校低很多），先怀疑是不是中外合作/民办二级学院/护理类单列组。
- 出表时标注每个组是否中外合作/高学费，尤其当用户已排除此类时。

### 陷阱 2：批次线/本科线数据错误（必核实）
任务书或用户口述的"本科线 XX 分"经常与官方不符。
- 反例（实战）：任务称"2025湖北物理本科线435"，**官方实为 426 分**（特殊招生线 516）。
- **必须独立核实当年官方批次线**，不直接采信任务书/用户口述。差异虽常不影响位次锚定结论，但要向用户澄清。

### 陷阱 3：文理/物理历史混淆
同一校同年，物理类和历史类录取线差异巨大。新高考省份务必锁定**考生对应科类**的数据。
- 反例：某校"504/83125"是历史类校线，物理类实际最低投档 482 —— 用错科类直接误判。

### 陷阱 3.6：一分一段表「总分混排」口径陷阱（定位考生位次必查）⚠️
一分一段表按 **普通类 / 艺术类 / 体育类 / 技能高考** 分多张发布；另有一张把所有类别混在一起的「**总分一分一段统计表**」。
- 铁律：**定位考生位次只能用「普通类（首选物理/历史）」专表，绝不能用含艺术/体育/技能高考的总分混排表。**
- 为什么：混排表把艺术/体育/技能考生也计入累计人数，同一分数对应的位次会**显著偏前**；而院校录取数据是按普通类排的，两端口径不一致就系统性误判档位。
- 反例（实战）：湖北 2026 物理类 546 分，普通类专表位次 = **64507**；某聚合站用总分混排表给出"546分=47563位"，偏前约 1.7 万名。若采信后者，会把一批根本够不到的院校误列为冲/稳档。
- 做法：取数时先看表头是否明写「普通类·首选物理/历史」；看到位次明显比常识偏前先怀疑是混排表，换省考试院/官方专表复核。

### 陷阱 3.5：跨省招生计划陷阱（考生省A×院校省B，强制核）⚠️
错误起源：拿院校「在本校所在省」或「在别省」的录取数据，去判断考生能不能考上。
- **铁律：录取线是「院校×省份」配对的，跨省不可比。** 考生在A省、报考B省的校，必须用**该校在A省**的录取线/位次/招生计划。
- 反例：某校在自己省录 600 分，在贫困远省可能只录 500 分；反之亦然。跨省拿数据=严重误判。
- **招生计划门槛（强制）**：该校若**本年不在A省投放某专业/专业组**，考生**根本报不了**，再合适也不能列入。报考前必查该校当年在考生省的分省分专业招生计划（省考试院发布的招生计划专辑 + 院校招生章程）。
- 出表的每一所院校，默认都是"该校在考生省有招生计划且已用该省数据推演"。

### 陷阱 4：分数 vs 位次混用跨年比较
绝不拿"去年某校录取 500 分"和"今年考生 500 分"直接比——分数线逐年漂移。**只比位次。**

### 陷阱 5：校线最低位次冒充好专业位次（高频致命，2026-06-26 实战血泪）⚠️⚠️
**校线最低位次 = 全校最冷门/压线组的位次；想读的好专业（计算机/AI/电气/临床等热门组）真实位次远高于此。** 用校线最低位次承诺"能进计算机/AI"，会让考生误判某校"稳"、实际想读的专业根本冲不上——这是最容易骗到自己的错误。

**铁律：每个专业组必须用「该专业组自己的真实录取位次」定档，绝不用校线最低位次冒充热门专业组。** 出表时，"能进这所学校"（用校线最低位次）和"能进这所学校的好专业"（用该好专业组位次）是两个不同结论，必须分别标注、用各自真实数据。

**实战反例（2026-06-26 湖北物理 91723 案例，发给家长前被交叉验证抓出，系统性错误）**：
- 武汉商学院：方案误把校线最低 88339（烹饪/食品组）挂到「人工智能/机器人」头上当稳档；**真实：AI=514分/74570位、机器人=510分/80494位、数据科学=513分/75381位**——对 91723 考生属冲不上的盲冲，差近 1.1万~1.7万名。
- 湖北理工学院：计算机/电气「国家级一流」真实在 07组514分/74570位、08组505分/82251位；方案却挂全校最低 106016（中外合作触底组），低估难度约 3 万名。
- 湖北工程/湖北文理/湖北民族大学：同样把"计算机/电气国一流"挂到全校最低位次，全是此陷阱。

**排查动作**：①看到"某校计算机国一流位次 = 该校全校最低位次"——必为此陷阱，立即用该专业组真实位次复核。②热门专业（计算机/AI/软件/电气/临床/财经）位次普遍比校线最低高 1万~4万名，若标得和校线最低一样低，几乎一定错了。③定档主语必须是"X校Y专业组"，不是"X校"。

### 陷阱 6：大小年/断档异常值
某年爆冷（断档）或爆热属异常。不用单年极值，用三年中位数 + 趋势判断。

### 陷阱 7：专项计划/地方专项混入普通组
国家专项、地方专项、高校专项的位次通常低于普通组，但有报考资格门槛（户籍/农村等）。考生不符合资格时，**这些组的低位次不能算作其能进的依据**。

### 陷阱 8：新增/合并/更名院校历史数据断层
院校更名、独立学院转设、专业组重组会导致历史位次不可直接沿用。遇到标注"历史数据断层，参考有限"。

## 数据采集纪律小结
- 一手官方 > 权威聚合 > 其他，能查官方绝不用二手。
- 每个位次：来源 + 年份 + 专业组性质，三者齐全。
- 关键位次 ≥2 源交叉。
- 查不到就说查不到。**宁可少推荐一所，不可填一个假数据。**
