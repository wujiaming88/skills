# 内容读取协议（所有研究任务共享）

## 最终输出硬规则（所有任务通用·最高优先级）
你的最终回复**必须以报告内容的第一个实义字符开始**（如 emoji 标题、结论句、或 HEARTBEAT_OK）。
回复开头**禁止**出现任何：
- 思考/过程旁白（如 "I now have..."、"Let me compose..."、"Now let me..."、"我现在来..."、"让我..."、"首先我..."）
- 动作/投递说明（如 "explicit target failed"、"final reply"、"报告已推送"、"已发送"）
- 读取量/工作量自述（如 "I've deep-read 5 articles"、"我读了X篇"）

所有思考放在工具调用阶段（运行命令/搜索/读取时），final reply **只放成品**。
投递由系统（cron announce）自动完成，**不要调用 message 工具主动发送**。

## 核心工作流

```
搜索/数据文件 → 线索（标题、URL、字段名）
        ↓ 这一步不产生结论
web_fetch / 数值解读 → 内容（正文段落、具体数值）
        ↓ 这一步才是研究
基于内容写分析 → 产出（每条判断可回溯到具体来源）
```

---

## 数据文件读取标准

读取 JSON/API 数据后，输出必须包含三层：

1. **具体数值**：原始数字（如 "南向净买入5.27亿港元"、"PMI 49.8"）
2. **对比维度**：环比/同比/vs预期/vs均值（如 "前值50.1，环比-0.3"、"较20日均值+2σ"）
3. **投资含义**：这个数值对决策意味着什么（如 "连续3月萎缩区间，制造业收缩趋势确认"）

---

## 文章/公告读取标准

对每篇 web_fetch 读取的内容，提取三类信息：

1. **关键原文引用**：原话/条款（如 "文件第三条明确提出'2026年底前完成整改'"）
2. **具体数据点**：文中提到的数字（如 "净投放1000亿，超出市场预期的500亿"）
3. **措辞/信号变化**：与之前表述的对比（如 "从'适度宽松'改为'更加积极'，政策温度上升"）

---

## 全文阅读硬门控（红线3·最高优先级）

**只读标题/摘要/搜索片段 = 没读。** 任何进入结论的判断，必须基于已 web_fetch / pdf 工具读到的**正文段落**。

硬规则：
1. 搜索结果（标题+snippet）只用来**选择要读什么**，绝不直接作为结论依据。
2. 每个关键判断背后，至少有一篇被**全文读取**的原文（财报/政策原文/研报/权威报道正文）。
3. 长文档（财报、招股书、政策全文、长研报）→ 用 pdf 工具或分段 web_fetch **读完相关章节**，不止读开头。摘要式略读不算数。
4. 读不到全文的源（付费墙/JS渲染）→ **诚实标注为信息盲区**，不靠标题脑补内容。
5. 一手 > 二手：能读财报原文就不引用新闻转述；能读政策原文就不引用解读稿。

## 读取量要求

| 任务类型 | 最低全文读取量 | 每条结论支撑 |
|----------|-----------|-------------|
| 宏观研究 | 8-12 篇正文 | ≥1 篇原文 |
| 政策研究/部委扫描 | 政策原文全文 + 5-8 篇解读 | 受益/受损判断 ≥1 手原文 |
| 行业深度研究 | 关键公司财报/招股书相关章节 + 行业报告 + 8-15 篇正文 | 每个分环节 ≥2 独立源 |
| 个股深度研究 | 最近财报全文 + 5-8 篇正文 | ≥2 篇/关键判断 |
| 投资可行性/风险分析 | 视主题，覆盖正反双方各 ≥3 独立源 | 每个情景 ≥1 数据支撑 |

读取量不是凑数指标——是"研究是否做透"的下限。不够就继续读，不靠减量换速度（用户明确：效果优先于速度）。

---

## 质量自检方法

推送前逐条检查你的输出：

1. 每个判断旁标注来源：`[来源: 财联社5/25 - "央行超预期净投放"]`
2. 标注不出来的判断 = 没有依据 → 删除或补充读取
3. 多源验证表格每个格子是否有具体数值？只写"确认"无数字 = 不合格

## 示例对照

### 宏观研究——合格输出
```
央行操作：今日开展7天逆回购2580亿元（中标利率1.4%），对冲到期10亿后净投放2570亿元，
连续4日净投放。[来源: RTHK 5/25 - "规模创近两个月单日新高"]
判断：跨月流动性呵护明确，"收短放长"格局下短端利率有下行空间。
```

### 宏观研究——不合格输出
```
央行进行了逆回购操作，市场关注流动性。
```
↑ 无具体金额、无期限、无来源引用、无对比、无判断依据

---

### 标的研究——合格的多源验证
```
- southbound: 确认 | 净买5.27亿港元，Top1，连续3日加仓，占日均成交15.3%
- analyst: 确认 | 高盛上调目标价至45.0（+12%），理由"28nm产能利用率达92%超预期"
- short_interest: 否认 | 沽空比例8.2%较上周+0.5%，空头小幅加仓
- announcements: 确认 | 5/20公告Q1收入同比+23%，毛利率38.2%（去年同期35.1%）
```

### 标的研究——不合格的多源验证
```
- southbound: 确认 | 南向有流入
- analyst: 确认 | 分析师看好
- short_interest: 无数据
```
↑ 无具体金额、无排名、无变化幅度、无分析师名称/理由
