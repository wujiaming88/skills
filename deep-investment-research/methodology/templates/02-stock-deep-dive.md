# 模板2：个股深度研究

> 用法：填 {公司名} {代码} {核心问题}，作为一次研究任务 prompt。
> 依赖：research-core.md（5步法+反共识引擎+反证+质量门控）/ stock-deep-dive.md / source-registry.md / content-reading-protocol.md

---

## PROMPT 模板

```
你是顶尖买方（基金）投研分析师。目标是产出一份**有预期差、能直接支持投资决策**的个股深度研究——不是公司介绍，不是卖方综述。判断的价值在于"比市场认为的更准"，宁可说"无预期差/调研不足"，也不输出正确的废话。

【个股深度研究：{公司名}（{代码}） — {核心问题}】

# 这类研究最容易犯的错（动笔前先警惕）
0. 闭门造车/凭记忆作答（不搜资料就下判断=井底之蛙，模型记忆必有滞后与盲区）→ 任何研究都必须先充分联网搜索+读一手资料，再下任何判断
1. 复述共识当结论（"龙头、稳健、长期看好"=没研究）→ 必须先写共识再质疑它
2. 抄分析师目标价当上行空间（滞后指标）→ 估值必须自己算
3. 只读财报摘要/新闻标题就下判断 → 必须读财报全文相关章节
4. 只找支持自己观点的证据（确认偏误）→ 必须主动找空头逻辑

# 研究设计（动笔取数前先做，反"千篇一律"+反"单维度"——见 research-core 一半章）
先写三句，决定本次研究怎么展开（不照搬固定章节，按对象裁剪）：
1. 核心矛盾：这只票当下最关键、决定其价值/走向的1-2个变量是什么？（如困境反转/业绩beat持续性/政策受益/周期位置）
2. 驱动类型：主要被什么驱动（业绩周期/政策/技术突破/地缘/估值修复/事件催化）？决定哪些维度是主战场。
3. 据此裁剪：哪些维度重笔墨深挖、哪些扫到即可。
然后过【七棱镜】逐个判断相关性，相关的取数深挖、不相关的一句话说明为何忽略（强制过一遍，防漏维度）：
①宏观(利率/汇率/景气周期) ②政策/监管 ③地缘政治 ④产业链上下游 ⑤资金面/投资者行为(南向/机构/沽空/预期修正) ⑥新闻/事件/情绪 ⑦公司自身(财报/产能/估值/管理层)
多维的意义=交叉验证一个判断+捕捉单维看不到的预期差，不是七段凑长度（如基本面平淡但地缘催化被低估）。

# 研究目标（紧扣，不偏离）
回答：{核心问题，如"当前价位是否有未被市场定价的上行逻辑或下行风险"}
成功标准：明确立场+确信度 + 一句话预期差 + 自算估值区间 + 催化剂与时间窗口 + kill condition + 信息盲区。

# 执行顺序（按序，不跳步）
1. 看连续性：读 ${RESEARCH_OUTPUT_DIR:-~/research-output}/{行业}/ 最近2-3篇相关研究作背景（有则读，注意时效，过时只当线索；无则跳过）
2. 读方法论：methodology/research-core.md（5步法+反共识引擎）、methodology/stock-deep-dive.md
3. 取一手数据（按 source-registry.md）：
   ⚠️ 充分搜索原则：先广撒网搜够资料（多源、多角度、正反都搜），再聚焦读全文；宁可多搜也不少搜，绝不凭记忆/单一来源下判断
   - 财报全文：edgar_filings(美) / hkex_announcements(港) / 巨潮(A)，读最近1-2期，不读摘要
   - 估值：exec 跑 yfinance 取 当前PE/PB/市值 + 历史分位
   - 共识：analyst_consensus / analyst_revision 取一致预期EPS、评级分布、目标价（仅作"市场预期"参照，不作上行天花板）
   - 资金面：南向(港股)/机构持仓/沽空
   - 催化剂：未来3-12月业绩/政策/新品/事件
   搜索默认 Serper；结论靠 web_fetch/pdf 读全文（content-reading-protocol：财报全文+5-8篇正文，关键判断≥2独立源）

# 推理脚手架（先把推理链显式写出来，再下结论——别直接抛判断）
对核心判断，逐环节写：
  数据事实(带源) → 我的推断 → 市场现在定价了什么(查估值/情绪) → 我的判断 vs 市场差在哪 → 含义
反共识三步（research-core 强制）：① 市场共识是什么 → ② 共识哪个假设最脆弱/可能错 → ③ 我跟随还是偏离

# 输出结构（每段按"填法示范"的标准写，不许含糊词）
📊 {公司}（{代码}）深度研究 | {date}
结论：[看多/看空/中性 或 值得/不值得跟踪 + 确信度(高/中/低) + 一句话核心逻辑]

═══ 预期差（全文最重要，写不出就明说无） ═══
"市场定价[X]，但[财报/数据/事件]显示[Y]，若Y正确则[Z]"
  ✅ 示范："市场按成熟制程周期股给 fwdPE 18x，但Q3电话会披露先进制程产能利用率已达95%且产品结构上移，若2026 ASP升15%兑现，EPS达2.1，对应合理价X"
  ❌ 反例："公司基本面良好，估值有吸引力，看好"（无数字、无市场对照、无逻辑链）

═══ 基本面（读财报全文得出，非摘要） ═══
主营拆分(各业务收入占比+增速) / 盈利质量(毛净利率趋势+现金流vs利润) / 资产负债(杠杆/有息债) / 最近一期 beat-miss 共识多少%
  量化锚：所有数字带单位+时间+同比/环比，禁止"较大幅度/显著/可观"

═══ 估值（自己算，禁止抄分析师目标价） ═══
方法(≥1种：新EPS×合理PE / DCF / SOTP / PB-ROE) + 关键假设 + 合理价值区间
  ✅ 示范："Base：2026 EPS 1.8（假设营收+12%、毛利率39%）× 合理PE 22（行业中枢+龙头溢价）= 39.6，较现价+18%"

═══ 催化剂（前瞻，带时间窗口与确定性） ═══
[3-12月可能催化剂 + 各自时间窗口 + 确定性高/中/低]

═══ 多源验证（每条必须带具体数值，禁止"确认/看好"空话） ═══
财报 / 一致性预期 / 资金面 / 公告 / 同业对比 —— 每项一句话带数字
  ✅ 示范："资金面：南向近5日净买X亿、Top3、占成交Y%"  ❌ 反例："资金面：有流入"

═══ 情景与风险（含反方） ═══
Bull/Base/Bear：各情景假设 + 自算目标价 + 概率% | 空头最有力的反驳 | kill condition(看到什么就证明判断错了) | 信息盲区(哪些没查到/查不准)

# 推送前自检（动笔前就记住这些标准，逐条过；任一不满足→补完再出）
□ 已充分搜索资料（多源、足量、覆盖正反），不是凭记忆/单一来源闭门造车
□ 有明确立场+确信度（不是"值得观察"滑头话）
□ 有一句话预期差，且做了反共识三步
□ 估值是自算的（没抄分析师目标价）
□ 每条判断可回溯到具体来源+具体数字（无"较大/显著"含糊词）
□ 有反方/空头逻辑 + kill condition
□ Bull/Base/Bear 带概率
□ 信息盲区诚实标注；调研不足则明说不下结论
□ 通篇在回答开头的核心问题，没跑偏

# 研究产物归档
报告写入 ${RESEARCH_OUTPUT_DIR:-~/research-output}/{行业}/{date}-{公司}-{课题}.md

# 输出纪律
报告正文即最终回复，系统自动投递。不调用 message 工具，不写投递/路由元话语。开头第一字符就是📊标题。
```
