# 主标题客观化规则与改写对照

## 三条标准（详解）

1. **客观**：标题陈述事实或主题本身，不带情绪煽动、不夸大价值判断。读者看完标题，对文章内容的预期与实际内容一致。
2. **准确**：标题里出现的每一个限定词（范围、程度、数量、时间）都能在正文找到对应支撑。不能"标题写全网最强，正文只对比了三家"。
3. **清晰**：一眼能看懂在讲什么。优先结构：`主题 + 范围/角度`，或 `主题：副标题补充范围`。

## 夸张词黑名单（主标题禁止出现）

颠覆、革命、碾压、屠杀、战争、王炸、炸裂、重磅、震撼、史诗级、终极、神器、一文讲透、彻底搞懂、外科手术、深挖、精准遗忘、xx 之死、封神、天花板、降维打击、遥遥领先、吊打、秒杀、最强、最牛、逆天、史上最、前所未有、改写历史、引爆、刷屏、爆火。

> 黑名单是底线，不是穷举。凡是为博眼球、超出正文内容支撑的情绪化表达，一律视同违规。

## 改写对照（真实病例 → 客观版）

| ❌ 夸张原标题 | ✅ 客观改写 | 问题 |

| AI Agent 的沙箱战争：从 chroot 到 Firecracker，一文讲透 Agent 安全地基 | AI Agent 的沙箱隔离：从 chroot 到 Firecracker 的技术演进 | "战争""一文讲透""地基"均夸张 |

| 深度剖析：Claude Code 的压缩引擎——源码究竟揭示了什么 | Claude Code 压缩引擎的源码分析 | "究竟揭示了什么"故弄玄虚 |

## 周报类标题

周报标题保持客观格式即可，已合格，不要加任何夸张修饰：
`{主题}研究周报 · 第 N 期（YYYY-MM-DD ~ MM-DD）`

例：`全球 AI Agent 基础设施研究周报 · 第 1 期（2026-06-11 ~ 06-17）` ✅

## 自检

拟完主标题后逐项问：
- [ ] 是否含黑名单词或同类情绪化表达？
- [ ] 标题每个限定词正文都能兑现吗（不夸大范围/程度）？
- [ ] 不看正文，读者能准确预期文章讲什么吗？
