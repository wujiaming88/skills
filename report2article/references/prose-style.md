# 文风：专业、不啰嗦、有人味

目标：**在不减少任何信息的前提下**，让文章读起来专业、干净、有一个真人编辑的声音，而不是一台罗列机器。

> 🔴 与红线的关系（必须先立住）：本文件所有手法只动**措辞、结构、强调方式**，绝不动**事实、数据、链接、结论强弱**。
> "精简"永远指**压缩表达**（phrasing），**绝不是压缩信息**（information）。删一个虚词可以，删一个事实不行。
> 判定：任何一处改写后，做红线比对——事实/数据/链接/判断强弱与原报告逐条等价，才算合格。

## 一、声音与节奏（Voice & rhythm）

- **长短句交替**：连续三句以上等长会像播报。用一个短句收束一段长论述，节奏就出来了。
- **主动语态优先**：`OpenAI 发布了 X`，而不是 `X 被 OpenAI 发布`。被动只在强调受事时用。
- **动词具体**：用"收购""开源""降价""合并"这类实义动词，少用"进行了……的动作""做出了……的举措"这类空壳。
- **一段一个主意**：一个自然段只讲一件事。塞三件事的段落要拆开。
- **编辑的声音**：像一个懂行的人在跟你讲清楚一件事，有判断的取舍（哪个先讲、哪个是重点），但**判断只能来自报告已有的结论**，不新增。

## 二、反啰嗦（Anti-verbosity）

**只删废话，不删事实。** 下列表达绝大多数不承载信息，删掉句子更利落：

- 起手空转：`值得注意的是`、`需要指出的是`、`不难发现`、`众所周知`。
- 万能收束：`总的来说`、`综上所述`、`总而言之`。
- 空泛趋势句：`随着……的发展`、`在……的大背景下`、`在当今时代`。
- 冗余修饰：`进行了一次深入的`→`深入`；`起到了……的作用`→直接说做了什么；`的相关`、`方面的`、`的过程当中`多数可删。
- 重复限定：一句里已经说了"本周"，后面每句不必再"本周"。

**方法**：先写完整，再逐句问"删掉这几个字，信息少了吗？"——没少就删。这是收紧措辞，不是丢信息。

## 三、优雅排版（Elegant formatting）

- **标题有目的**：每个小标题是路标，指向下面这段讲什么，不是装饰。（细则见 `title-rules.md`）
- **加粗克制**：只给**真正的关键术语/关键数字**加粗，一段最多一两处。满屏加粗等于没有重点。
- **列表 vs 叙事**：叙事、演变、因果用**段落**；只有**真正平行、可枚举**的项（几家厂商各自的动作、几个并列特性）才用列表。不要把有逻辑关系的内容拆成无主语的 bullet。
- **留白与导语**：开篇一段独立的导语（lead）交代"这期讲什么、为什么值得读"；段落之间留呼吸感，不堆成墙。
- **点睛不滥用**：一句提炼式的小结或引句，只在它真的点睛时用一次，别每节都来一句。

## 四、人味（Human feel）

- **强开头**：第一段就回答"这为什么重要"，用报告里最有分量的事实/结论切入，别用"随着 AI 的发展"开场。
- **连接组织**：用承上启下句把孤立的点连成一条线（四段骨架帮你排线）；过渡句只导航，不夹带新信息（见红线）。
- **收得住**：结尾**综合**全文给读者一个落点（报告已有判断的收拢），而不是写着写着断掉或用"让我们拭目以待"敷衍。

## 五、Before / After（信息 100% 保留，只改措辞与结构）

**❌ 啰嗦 / 机械版：**

> 值得注意的是，随着 AI Agent 领域的不断发展，本周谷歌方面进行了一次重要的产品发布动作。谷歌在本周发布了 Vertex AI Agent Builder。据相关报道，该产品的定价方面是每千次调用收费 2 美元。总的来说，这一举措起到了推动 Agent 生态发展的作用。

问题：`值得注意的是`/`随着……的发展`/`方面`/`进行了一次……动作`/`总的来说`/`起到了……的作用`全是空转；三句话拆成了一堆短碎句还互相重复"本周""谷歌"。

**✅ 收紧 / 优雅 / 有人味版：**

> 本周谷歌发布了 Vertex AI Agent Builder，定价每千次调用 2 美元。报告判断，这会推动 Agent 生态。

核对：三个事实（谷歌发布、产品名 Vertex AI Agent Builder、定价 2 美元/千次调用）+ 一条报告已有判断（推动 Agent 生态），**一个不少**；`每千次调用 2 美元`这个数字原义保留；只删了废话、合并了重复主语。若报告原文附了链接，仍须在文中保留。

> ⚠️ 反面警示：如果为了"精炼"把"定价每千次调用 2 美元"删成"定价合理"，或把报告的"可能推动"写成"必将推动"——**这不是文风优化，是越红线**，一律打回。
