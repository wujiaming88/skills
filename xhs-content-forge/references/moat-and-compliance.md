# 护城河(反抄袭) + 平台合规红线

> 自检阶段读。两条独立闸门:① 这篇有没有"抄不走"的东西;② 有没有踩平台红线。

---

## 一、护城河四维(每篇 ≥2 项)

核心自检问:**"这篇内容换个 AI 神器号能不能照抄?"** 能 → 壁垒没体现,必须加真实团队专属元素再发。

| 维度 | 怎么在内容里体现 |
|------|------------------|
| **① 真团队 × 时间沉淀** | 埋"时间锚点":这是五岳第 N 周 / 上周三泰山部署的那个服务今天出问题了。时间线本身就是壁垒,假团队补不上前情。 |
| **② 连载叙事** | 接主线:黄山上周调研的方向这周出报告了 / 华山的战略上月被推翻重做。结尾留"下集预告"。 |
| **③ 人格化 IP** | 让某个专家"有性格"地出场:黄山又抠细节、华山又画饼、泰山闷头干完甩一句话。养评论区"催更人格梗"。 |
| **④ 真实数据 & 翻车** | 晒可追溯的真实产出/数据,或真实翻车现场(AI 干砸了啥、我怎么修)。假号不敢晒翻车。 |

每篇发布前勾选(来自作战手册 12.3):
- [ ] 有 ≥1 个只有真团队才知道的独家细节(具体时间/报错/协作对话)?
- [ ] 接上了主线连载(能关联前后集,新人补不上前情)?
- [ ] 体现了团队人格/黑话(某专家性格梗、团队关系)?
- [ ] 有可追溯的真实证据(时间锚点/里程碑/真实数据或翻车)?
- [ ] 以上全没有 → 这是"任何人都能发"的通稿,**重写**,至少加两项。

一句话:**工具号能复制方法,复制不了一支真跑了几个月、有性格、有翻车、有连载剧情的 AI 团队。**

## 二、合规红线(绝不违规)

平台 2025-2026 在严打:低质 AIGC、批量洗稿、虚假人设、站外导流。我们靠"真实纪实"顺规则,但操作上仍要逐条照做。

### 🚫 绝对不做(碰即限流/封号)
- 私信/正文/评论直接发微信号、手机号、二维码
- 图片上 P 微信号(OCR+AI 能精准识别)
- 谐音变体(VX / 威信 / V❤️,模型已能读懂)
- 头像/昵称/简介带第三方平台信息或诱导语("关注后分享""私信领取")
- 大小号配合建群导流、矩阵号发同质化内容
- 编造数据、夸大成果、虚构案例(违背真实纪实定位,且踩虚假人设线)
- 标题用极限词:最/第一/100%/唯一,以及医疗功效词

### ✅ 只走这些合规引流路径
1. **内容钩子 + 主动私信**:正文结尾引导"想要完整 SOP → 评论区聊 / 私信我",靠内容价值让用户主动来,**不留任何联系方式**。
2. **企业号 + 私信通**(官方唯一合规导流):用官方组件(欢迎语/留资卡)收集需求。
3. **站内承接**:需求沉淀到置顶笔记 + 笔记合集,先在站内做轻咨询。
4. **AIGC 标识合规**:AI 辅助生成/润色的笔记,发布页"高级选项→内容类型声明"勾选"已自主标注"。本账号以真实纪实为主,凡 AI 生成图文如实标注,零侥幸。

### 合规自检(每篇)
- [ ] 正文/配图/评论里没有任何联系方式或谐音变体?
- [ ] 没有极限词、医疗功效词?
- [ ] 数据和成果都真实、可追溯,没有夸大?
- [ ] 涉及 AI 生成的图文,提醒用户发布时勾选 AIGC 声明?
